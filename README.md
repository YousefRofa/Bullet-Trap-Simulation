# Bullet-Trap-Simulation

To use the NANOBEM toolbox, you must add at the beginning of each session
the main directory and all subdirectories to the Matlab path, e.g. by calling

> addpath(genpath(NANOBEMDIR) );

Here, NANOBEMDIR is the *full* directory name of the toolbox.  

For the Bullet simulation, you should run the file /demo/tweezer/demobullet01.m

Before running it, you have to initialze the bullet shape by running demotmat01.m

Make sure that the equation used to describe the shape of the bullet in both the
demotmat01 and the demobullet01 match before running the simulation.
