BoundaryEdge

BoundaryEdge is a class derived from the superclass BoundaryElement that stores the discretized boundary elements together with Raviart-Thomas shape elements.

Contents

  • Initialization
  • Methods
  • Examples

Initialization

%  set up array of boundary elements
%    mat    -  array of Material objects
%    p      -  particle objects
%    inout  -  index to material at inside and outside
%   obj     -  vector of boundary elements
obj = BoundaryEdge( mat, p1, inout1, p2, inout2, etc );

After initialization, the BoundaryEdge elements hold in addition to the properties of the BoundaryElement class the following elements

  • obj.nu global degree of freedom for edges of boundary element
  • obj.val prefactor for Raviart-Thomas shape element, see Hohenester, Nano and Quantum Optics (Springer 2020), Eq. (11.36)

Methods

In addition to the methods of the BoundaryElement class, the BoundaryEdge class has the following methods

%  Raviart-Thomas shape element and its divergence, as well as positions
%    X,Y are triangular coordinates
[ f, fp, pos ] = basis( obj, x, y );
%  number of global degrees of freedom
n = ndof( obj );
%  plot value or vector array
plot( obj, varargin );
%  computation of inhomogeneity for Galerkin scheme
%    FUN is function that returns incoming fields for given Point positions
[ qe, qh ] = qbasis( obj, fun );
