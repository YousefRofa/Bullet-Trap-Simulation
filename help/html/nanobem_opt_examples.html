Examples for +optics classes

In the following we list the demo files for the +optics features of the nanobem toolbox.

  • demooptics01.m - Focusing of incoming fields.
  • demooptics02.m - Imaging of point dipoles using optics.lensimage.
  • demooptics03.m - Imaging of point dipoles, reversed optical axis.
  • demooptics04.m - Imaging of point dipoles using optics.lensimage2.
  • demoiscat01.m - iSCAT for various nanoparticles.
  • demoiscat02.m - iSCAT for gold nanosphere and varying focus planes.
  • demoiscat03.m - Comparison iSCAT and simplified dipole model.
  • demoiscat04.m - iSCAT with quarter wavelength plate.
  • demoiscat05.m - iSCAT images with interactive slider.
  • demoiscat06.m - iSCAT for comparison with Mahmoodabadi et al., Fig. 3b.
  • demoiscat07.m - Same as demoiscat02.m but with OPD abberation.
  • demofield01.m - Electric field maps for different setups.
  • democorr01.m - Fit best focus plane to experimental data.
  • demofocus01.m - Focus fields for an incoming Laguerre-Gauss beam.
